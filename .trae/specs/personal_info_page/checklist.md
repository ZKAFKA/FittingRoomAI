# 个人信息页面 - 验证检查清单

## 功能验证
- [ ] 从settings页面点击个人信息入口能成功跳转到个人信息页面
- [ ] 页面加载时能正确显示用户当前信息（头像、用户名、性别、身高、体重）
- [ ] 点击头像区域能弹出选择菜单（从相册选择/拍照）
- [ ] 上传新头像后能成功更新并显示
- [ ] 用户名输入框能正常输入和修改
- [ ] 性别选择按钮能正常切换并显示选中状态
- [ ] 身高体重输入框能正常输入数字
- [ ] 点击保存按钮能成功保存所有修改的信息
- [ ] 保存成功后能显示提示信息
- [ ] 点击返回按钮能成功返回上一页
- [ ] 未登录时点击个人信息入口能显示登录授权提示
